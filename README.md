# 提词器

一款基于 PyQt5 开发的专业提词器应用程序，适用于直播、演讲、会议等多种场景。

## 功能特性

### 核心功能
- **高性能文本渲染** - 支持大文本流畅显示和滚动
- **平滑滚动系统** - 基于定时器的平滑滚动，速度可调
- **智能分段管理** - 支持自动识别段落标识，精确控制每段停留时间
- **多屏幕扩展** - 支持主副屏同步显示，适合多显示器环境
- **独立控制面板** - 集成所有功能控制，操作便捷

### 详细功能

#### 文本管理
- 打开/保存/另存为文本文件
- 实时编辑提词内容
- 支持使用 `({分:秒})` 格式标识段落和停留时间
- 自动恢复上次打开的文件

#### 滚动控制
- 开始/暂停/重置滚动
- 可调节滚动速度（支持非线性速度调节）
- 实时显示滚动时间信息
- 支持段落内滚动进度控制

#### 段落设置
- 自动分段识别
- 全局段落停留时间设置
- 局部段落自定义停留时间
- 自动段落跳转功能
- 段落导航（上一段/下一段）

#### 样式定制
- 字体大小调节
- 背景颜色自定义
- 文本颜色自定义
- 实时预览效果

#### 多屏设置
- 主副屏同步显示
- 窗口位置和大小精确调整
- 独立置顶设置
- 滚动位置实时同步

#### 配置管理
- 配置持久化保存
- 支持重置为默认配置
- 自动保存窗口状态

## 安装

### 环境要求
- Python 3.7+
- Windows 操作系统

### 依赖安装
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

## 使用说明

### 基本操作
1. 在控制面板的"文本管理"标签页中输入或粘贴提词文本
2. 使用 `({分:秒})` 分隔不同段落，例如：
   ```
   欢迎来到直播间
   ({0:30})
   今天我们要介绍的是一款新产品
   ({1:00})
   ```
3. 在"滚动控制"标签页中调节滚动速度
4. 点击"开始"按钮开始滚动
5. 点击"暂停"按钮暂停滚动
6. 点击"重置"按钮重置滚动位置

### 高级功能

#### 段落时间控制
- **全局模式**：所有段落使用统一的停留时间
- **局部模式**：使用段落标识中的自定义时间

#### 多屏显示
1. 在"多屏设置"标签页中启用副屏
2. 调整副屏位置和大小
3. 可独立设置主副屏置顶

#### 配置保存
- 点击"保存配置"按钮保存当前设置
- 点击"重置配置"恢复默认设置
